# EVY Web

## Getting Started

### Installing Dependencies

```bash
deno install
```

## Running the App

### Locally

```bash
deno run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### In Docker

```bash
docker compose up -d
```
